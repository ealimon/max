# MAX | AI Intake Specialist for Limon Media

MAX is a custom-built AI agent designed to bridge the gap between business owners' goals and technical marketing strategies. Built for [Limon Media](https://www.limon.media/), MAX engages website visitors in a consultative dialogue to identify their needs—whether it's more phone calls, increased foot traffic, or higher e-commerce sales.

## 🚀 Features
- **Goal-Oriented AI:** Powered by Google's Gemini 1.5 Flash to translate business pain points into marketing solutions (SEO, Google Ads, Web Design).
- **Streamlit Interface:** A clean, modern chat interface tailored for the Limon Media brand.
- **Lead Qualification:** Automatically guides users toward providing essential contact information and business details.

## 🛠️ Tech Stack
- **Python**
- **Streamlit** (Frontend & Hosting)
- **Google Generative AI SDK** (Gemini API)

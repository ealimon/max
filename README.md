# MAX | AI Intake Specialist 🤖

**MAX** is a custom-built AI Agent designed for **Limon Media**. It serves as a professional first point of contact for potential digital marketing clients, providing a seamless, automated intake experience.

## 🚀 Overview
MAX lives on the front lines of client acquisition. Built using **Python**, **Streamlit**, and the **Gemini 1.5 Flash** model, MAX is designed to qualify leads, answer agency-related questions, and prepare prospects for a human follow-up.

## ✨ Key Features
* **Lead Qualification:** Automatically identifies client needs (SEO, PPC, Web Design).
* **Persona-Driven:** Programmed with the professional and energetic voice of Limon Media.
* **Memory-Aware:** Remembers context throughout the conversation for a natural feel.
* **Streamlined Handoff:** Prepares prospects for direct consultation with Edward Limon.

## 🛠️ Technical Stack
* **Frontend/Backend:** [Streamlit](https://streamlit.io/)
* **AI Engine:** [Google Generative AI (Gemini)](https://ai.google.dev/)
* **Security:** Managed via Streamlit Secrets (No API keys stored in repo).

## 📂 Project Structure
* `app.py`: The core logic and AI persona.
* `requirements.txt`: Necessary libraries for deployment.
* `.gitignore`: Security protocols to protect private configurations.
